# Text to Speech Generation using Hugging Face

This project demonstrates how to generate speech from text using a pretrained **Text-to-Speech (TTS) model** from the Hugging Face Transformers library.

The notebook converts a given text input into an audio waveform and plays the generated speech directly inside a Jupyter notebook.

---

# Project Overview

Text-to-Speech (TTS) is a technology that converts written text into spoken audio. In this project, a pretrained model from Hugging Face is used to generate speech from text input.

The workflow of this project includes:

1. Installing required libraries  
2. Loading the Hugging Face pipeline  
3. Generating speech from text  
4. Displaying and playing the generated audio  

---

# Technologies Used

- Python
- Hugging Face Transformers
- Pretrained TTS Model (Bark)
- Jupyter Notebook
- IPython Display

---

# Model Used

The project uses the **Bark Small** model from Hugging Face.

**Model Name:** `suno/bark-small`

Bark is a generative audio model capable of producing natural sounding speech from text input.

---

# Workflow

## 1. Install Required Libraries

The Transformers library is installed to access pretrained models and pipelines from Hugging Face.

---

## 2. Import the Hugging Face Pipeline

The Hugging Face `pipeline` API provides an easy way to use pretrained models for tasks such as:

- Text generation
- Image generation
- Speech recognition
- Text-to-speech

In this project, the pipeline is used for the **text-to-speech task**.

---

## 3. Provide Text Input

A text sentence is defined as input.  
This text will be converted into spoken audio using the pretrained model.

Example text:

Python is a high level, general purpose programming language.

---

## 4. Load the Text-to-Speech Model

The pipeline loads the **Bark Small model**, which is designed for speech generation.

The model processes the input text and produces an audio output.

The output generated by the model contains:

- **audio** – the generated speech waveform
- **sampling_rate** – the number of samples per second used for audio playback

---

## 5. Generate Speech from Text

The text is passed to the pipeline, which processes it and generates the corresponding audio waveform.

This step performs the core **text-to-speech generation**.

---

## 6. Play the Generated Audio

The generated audio is played directly in the notebook using **IPython.display**.

An audio player widget appears inside the notebook, allowing the user to listen to the generated speech.

The audio player uses:
- the generated audio data
- the sampling rate for correct playback

---

# Output

The final output of the project is:

- Generated speech from the input text
- An audio player embedded in the notebook to listen to the generated speech
